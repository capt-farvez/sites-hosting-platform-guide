# Sites Hosting Platform Guide

A guide repository for BSS interns covering the deployment process for websites and web applications across various hosting platforms.

## What You'll Learn

- How to deploy sites using **Netlify**
- How to deploy sites using **Vercel**
- Other hosting platforms and deployment strategies

## Who Is This For?

This repository is intended for **BSS (Bangladesh Software Solution) interns** who are getting started with web deployment. It walks through the setup, configuration, and deployment steps for each platform so you can confidently ship your projects to production.
